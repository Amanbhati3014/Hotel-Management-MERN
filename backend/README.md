# 🏨 Hotel Management System (MERN Stack)

A full-stack Hotel Management System built using the MERN stack to simplify hotel room booking and administration.

This platform allows users to browse hotel rooms and enables administrators to manage room listings, pricing, and room details.

---

# Features

## User Features

* User Registration & Login
* Secure Authentication
* Browse Available Rooms
* View Room Details
* Responsive UI

## Admin Features

* Add New Rooms
* Upload Room Images
* Update Room Information
* Delete Rooms
* Manage Room Listings

---

# Tech Stack

Frontend:

* React.js
* Axios
* React Router

Backend:

* Node.js
* Express.js

Database:

* MongoDB
* Mongoose

Authentication:

* JWT
* bcryptjs

Image Upload:

* Multer

---

Run backend:

npm run dev

Run frontend:

npm run dev
